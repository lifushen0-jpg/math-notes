# 1. 结论

> [!note] 几个公式
> - $$|I-AB| = |I-BA|$$
> - 行列式降阶公式：对可逆的矩阵 A , D，有$$|A||D-BA^{-1}C|=|D||A-CD^{-1}B|$$
> - 秩的降阶公式：对可逆的矩阵A,D，有$$r(A)+r(D-BA^{-1}C)=r(D)+r(A-CD^{-1}B)$$
> - 对矩阵 A，B，若$I_n+AB$可逆，则$$(I_n+BA)^{-1}=I_n-B(I_n+AB)^{-1}A$$
> - 对向量  $\alpha$，$\beta$，若矩阵 $A+\alpha\beta^T$ 可逆，则其逆矩阵为：$$A^{-1}-\frac{A^{-1}\alpha\beta^{T}A^{-1}}{1+\beta^T A^{-1}\alpha}$$这样可得：当 A 可逆且 $1+\beta^T \alpha \not= 0$时矩阵 $A+\alpha\beta^T$ 可逆
# 2. 矩阵的迹
## 迹是什么？

矩阵的迹是对角线元素之和，记为tr(A)
## 矩阵的迹的性质 
1.  $tr(A+B)=tr(A)+tr(B)$
2.  $tr(kA) = k\times tr(A)$
3.  $tr(AB) = tr(BA)$
4.  $tr(A) = tr(A^T)$
5.  $tr(AA^T) \geq 0$
6.  $tr(A^{-1}BA) = tr(A)$

## 矩阵的迹在矩阵运算中的作用

### 例1 ：  
证明：不存在方阵A、B， 使 $AB-BA = I_n$

proof:
对目标式两边求迹，有：
$$ n = tr(A) = tr(AB - BA) = tr(AB) - tr(BA) = tr(AB) - tr(BA) = 0$$
